# tmux

Tmux config source.
Start: `tmux.conf`, then `mappings/`.
Avoid editing `tmux-open/` unless required.

## Development

See [DEVELOPMENT.md](./DEVELOPMENT.md) for copy-mode keybinding gotchas and
debugging notes (run-shell env, `#{pane_id}`, popup lifecycle).

## Plugins

- `tmux-intray` — status panel, run from `~/other/tmux-intray/`
- `tmux-open/` — vendored plugin (avoid editing unless required)
